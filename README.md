# pharmacology2254

Pharmacology and bioinformatics analysis workflows (e.g. RNA-seq/scRNA-seq processing, differential expression, pathway analysis).

## Getting Started

1. Clone this repository.
2. Add your analysis scripts under a directory named for the dataset or project (e.g. `GSE12345/`).
3. Keep raw data files out of version control — see `.gitignore`.

## Contributing

- Open a pull request for any change; keep PRs small and focused.
- Describe the "why" behind a change in the PR description.
